# Sunny Seas Restaurant Website This project is an elegant, responsive website for "Sunny Seas" restaurant located in a small sunny town, catering to international tourists. The website highlights the restaurant's exquisite dishes, local wine and cheese, and tranquil setting. ## Features - Beautiful hero section with background image - About section with restaurant story - Menu showcase with images of dishes - Experience section describing the ambiance - Contact form for reservations and inquiries - Social media links ## Files - `index.html` - Main webpage structure - `styles.css` - Styling for the website - `images/` - Folder containing images (see below) - `favicon.ico` - Favicon icon ## Setup 1. Download or clone this repository. 2. Ensure the `images/` folder contains relevant images: - `sunny-town.jpg` - `dish1.jpg`, `dish2.jpg`, `dish3.jpg`, `dish4.jpg` - Social icons: `facebook.svg`, `instagram.svg`, `tripadvisor.svg` 3. Open `index.html` in your web browser to view the website. ## Customization - Replace images with your own for a personalized look. - Update contact details in the contact section. - Adjust menu items as needed. ## License This project is open source and free to use.
